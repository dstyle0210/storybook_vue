# vuePlayground : VUE + StoryBook

Vue3 + Vite + Typescript + StoryBook 으로 만드는 Vue 프로젝트 스케폴딩

## IDE 셋팅 ( Recommended IDE Setup )
[VSCode](https://code.visualstudio.com/) + 
[Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar) (and disable Vetur) + 
[TypeScript Vue Plugin (Volar)](https://marketplace.visualstudio.com/items?itemName=Vue.vscode-typescript-vue-plugin) +
[Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)

## 프로젝트 셋팅 ( Project Setup )

```sh
npm install
```

### VUE 개발 실행 (Compile and Hot-Reload for Vue Development )

```sh
npm run dev
```

### VUE Build 실행 ( Vue Type-Check, Compile and Minify for Production )

```sh
npm run build
```

### StoryBook 개발 실행 (Compile and Hot-Reload for StoryBook Development )
```sh
npm run storyBook
```

### StoryBook 빌드 실행 ( Storybook Compile and Minify for Production )
```sh
npm run storyBook-build
```
